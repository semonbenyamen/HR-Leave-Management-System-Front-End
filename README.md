# HR Leave Management System

## 📌 Overview

The **HR Leave Management System** is a centralized web application that allows employees to manage leave requests بسهولة، ويتيح للـ HR متابعة الطلبات والأرصدة بدل استخدام ملفات Excel التقليدية.

## 🚀 Features

### 🔐 Phase 1: Authentication & User Management

* User Registration & Login
* JWT Authentication
* Role-based access (Employee / HR)

---

### 📊 Phase 2: Dashboard & Leave Balance

* User Dashboard
* Leave Balance Overview
* Year-to-date leave summary

---

### 📝 Phase 3: Leave Requests

* Apply for leave
* Automatic calculation of leave days
* Validation based on available balance

---

### 🏢 Phase 4: HR Management

* View pending requests
* Approve / Reject requests
* Leave history tracking

---

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/semonbenyamen/HR-Leave-Management-System-Front-End.git
```

### 2. Navigate to project folder

```bash
cd HR-Leave-Management-System-Front-End
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the project

```bash
npm run dev
```
